# mydrumkit.lv2

## Visão Geral
`mydrumkit.lv2` é um plugin de kit de bateria para Linux (LV2), desenvolvido para oferecer uma solução leve, funcional e estável para uso em gravação, arranjo e ensino.

<img width="1366" height="695" alt="image" src="https://github.com/user-attachments/assets/62393cb9-61e3-4212-b4e9-4a24dd4251fb" />

## Funcionalidades  
- Suporte a múltiplos samples de bateria acústica.
- Integração com hosts(DAW) LV2.
- 6 Round Robins.
- 1 velocity Layer.

## Outputs (saídas de áudio separadas)
1. Kick
2. Snare
3. Hihat
4. Snare FX
5. Racktom 1
6. Racktom 2
7. Racktom 3
8. Floortom 1
9. Floortom 2
10. Floortom 3
11. Overhead L
12. Overhead

## Requisitos  
- Sistema operacional Linux.
- Host de plugins compatível com LV2 (ex: Carla, Qtractor, Ardour, REAPER com suporte LV2). 

## Instalação
1. Baixe aqui [AQUI](https://github.com/samuelsantanaoficial/mydrumkit.lv2/releases/latest):
2. Copie/mova a pasta `mydrumkit.lv2` para `~/.lv2`.
3. Carregue o plugin no seu host LV2.
